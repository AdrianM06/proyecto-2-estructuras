Nosotros somos Adrián Martínez, Gianfranco Camporeale y Alirio Ugarte; y aquí le presentamos nuestro proyecto 2 de estructuras de datos de este trimestre.

Aquí, podra simular una cola de una impresora, en el cual podrá agregar usuarios manualmente o cargándolos mediante un archivo, también podrá elegir que tan prioritario es la persona que se agrega; podrá a su vez crear varios documentos de cada usuario y mandar los documentos que desee a la cola de impresión, donde pudiera elegir cuales documentos necesitan ser impresos inmediatamente o si necesita eliminar espacio en la impresora.
